Recomendações

Antes de executar

1

Verifique se a lixeira não contém arquivos importantes antes de executar a limpeza.

2

Evite rodar durante atualizações do Windows em andamento para prevenir conflitos.

3

Após a limpeza, a primeira abertura de alguns programas pode ser levemente mais lenta — o Windows reconstrói os caches automaticamente.

4

Para possível rollback futuro, faça backup antes de executar (a pasta Windows.old será removida).
